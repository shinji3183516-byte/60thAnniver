RUN-CAR 背景移動 v3 修正内容

原因:
- style.css内に .era-bg-layer { transform: translateX(0) !important; } が残っており、CSSのkeyframes側で指定したtransformが効かない状態でした。
- そのため、背景の年代切替は動いても、横方向の流れは見えませんでした。
- CSSアニメーションだけで直すと、過去の !important 指定と競合しやすいため、今回はJSの requestAnimationFrame でCSS変数 --run-bg-x を直接更新する方式に変更しました。

修正:
- 年表停止中も背景は止まらないよう、年表のanimate()とは別のRAFで背景を動かします。
- 背景は左から右へ常時移動します。
- つなぎ目は許容し、走行感を優先しました。
- 1979年までは昼固定。
- 1980年以降は120秒サイクル。
- 夜は40秒から60秒へ延長。
- 車の上下ふわふわは止めました。
